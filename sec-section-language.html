    Section 1.1 The language of set theory

    Our version of the language of set theory will consist of a very few symbols together with some basic rules for how these symbols may be concatenated. The language incorporates the following symbols:

    Definition 1.1.1.

    The language of set theory, denoted \(\lst\text{,}\) consists of the following symbols:
    connectives \(\quad\wedge\) and \(\neg\)
    quantifier \(\quad\forall\)
    variables \(\quad v_0, v_1, v_2, \dots\text{,}\)
    parentheses \(\quad (\quad\) and \(\quad)\text{,}\)
    the equality symbol \(\quad=\text{,}\) and
    the membership symbol \(\quad\in\text{.}\)
    Given our list of allowed symbols, the meaningful statements that can be expressed in our language are produced as follows:

    Definition 1.1.2.

    1. An expression in the language of set theory is defined to be any finite string built from the symbols of \(\lst\)
    2. We define a formula of the language of set theory to any expression generated using the following rules:
      • \((v_i\in v_j)\) and \((v_i=v_j)\) are formulas (called atomic formulas) for any \(i\) and \(j\)
      • If \(\phi\) and \(\psi\) are formulas then so are \((\phi\wedge\psi)\) and \((\neg\phi)\)
      • If \(\phi\) is a formula then so is \((\forall v_i\phi)\) for any \(i\text{.}\)
    3. We let \(\Form_{\lst}\) denote the collection of formulas of \(\lst\) defined above.
    The above is our official way of building formulas. We will shortly introduce the other standard logical symbols as abbreviations of more complex formulas from \(\Form_{\lst}\text{,}\) but as we expand our language we want to keep in mind that the expressions and formulas in the augmented language are actually abbreviations of something written in \(\lst.\) Our official language is more limited so that it is easier to make definitions and prove facts about \(\Form_{\lst}\) without needing to break into too many cases. We illustrate the way this works with a couple of definitions and a lemma.

    Definition 1.1.3.

    By recursion we define a function \(\subform\) that assigned to each formula \(\phi\) of \(\lst\) the set \(\subform(\phi)\) of all of its subformulas
    • If \(\phi\) is an atomic formula, then \(\subform(\phi)=\phi\)
    • \(\displaystyle \subform((\phi\wedge\psi))=\subform(\phi)\cup \subform(\psi)\cup\{(\phi\wedge\psi)\}\)
    • \(\displaystyle \subform((\neg\phi))=\subform(\phi)\cup\{(\neg\phi)\}\)
    • \(\displaystyle \subform((\forall v_i\phi))=\subform(\phi)\cup\{(\forall v_i\phi)\}\)
    The above function takes as input a formula \(\phi\) of\(\lst\) and returns the collection of all the formulas of \(\lst\) that are used to build \(\phi\text{.}\)

    Definition 1.1.4. Free and Bound.

    1. The range of an occurence of a quantifier \(\forall v_i\) in a formula \(\phi\) is the unique subformula \((\forall v_i\psi)\) of \(\phi\) beginning with the occurence in question.
    2. An occurence of a variable \(v_i\) in a formula is bound if it is within the range of a quantifier of the form \(\forall v_i\text{.}\) Otherwise the occurence is free.
    3. A formula without free variables is called a sentence, and we let \(\Sent_{\lst}\) denote the collection of sentences in our language \(\lst\text{.}\)
